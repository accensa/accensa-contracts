# Security Policy

## Supported Versions

Currently only the `main` branch is supported for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in Accensa, please DO NOT report it publicly. 
Instead, reach out to the maintainer via email (placeholder for contact) or via direct message in the Stellar Developer Discord.

Please allow 48 hours for a response and triage.

**Note:** The smart contracts in this repository are currently UNAUDITED. Use them at your own risk.
